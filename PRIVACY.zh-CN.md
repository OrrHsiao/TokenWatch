# TokenWatch 隐私政策

[English](./PRIVACY.md)

生效日期：2026 年 7 月 4 日

TokenWatch 是一款仅在本地运行的 macOS 应用，用于汇总你 Mac 上 coding agent 记录中的 token 用量。

## 数据收集

TokenWatch 不收集、传输、出售、分享或上传用户数据。

TokenWatch 不需要账户或登录。它不包含分析、广告、遥测、追踪或第三方 SDK。

## 本地文件访问

TokenWatch 只会在你通过 macOS 文件选择器授权后，读取本地 Claude Code、Codex 和 opencode 使用记录。所有解析、汇总和费用估算都在你的设备本地完成。

应用可能会显示从这些文件中得到的本地信息，例如 token 数量、模型名称、会话标识符和项目路径。这些信息会保留在你的设备上。

## 本地存储

TokenWatch 会在本地 UserDefaults 中保存应用偏好设置和 security-scoped bookmark，以便记住设置并恢复你已授权文件夹的访问权限。这些数据只存储在你的设备上，不会被传输到任何地方。

## 网络访问

TokenWatch 应用本身不访问网络。“隐私政策”入口会使用你的默认浏览器打开公开的隐私政策页面。
