# Security

TokenWatch reads local coding-agent usage data after the user grants macOS file access. Please do not post private prompts, responses, usage records, credentials, or full local project paths in public issues.

## Reporting

For non-sensitive bugs, open a GitHub issue with a minimal reproduction.

For sensitive security concerns, use GitHub private vulnerability reporting if it is available for this repository. If it is not available, open a minimal public issue that says you have a private security report, but do not include exploit details or private data.

## Supported Versions

The default branch is the actively supported development version.
